Git常用指令

  • git init：在当前目录下创建git仓库
  • git add filename ：添加文件到暂存区
  • git commit：提交文件到本地仓库
  • git clone：从远程克隆仓库到本地
  • git push [remote] [branch]： 同步本地指定分支到远程仓库
  • git fetch：从远程获取最新版本到本地，不会合并代码(merge)
  • git pull：从远程获取最新版本并合并代码

    注意事项

  • TortoiseGit下载地址: https://tortoisegit.org/download/
  • Github同步有两种方式,使用ssh同步时需要先在本地生成密钥对——公钥与私钥。
    使用ssh-keygen -t rsa命令在本地生成密钥对,将公钥复制粘贴到Github->Settings->SSH and GPG keys->New SSH key。
  • 使用TortoiseGit工具的ssh方式同步时，由于TortoiseGit默认使用的ssh工具是putty需要在settings->network->ssh修改为GitBash的ssh工具
  • 不小心删除工作区的本地文件未提交时可以通过TortoiseGit工具的revert功能恢复。
