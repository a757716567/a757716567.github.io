hexo

常用命令

  • hexo init 命令用于初始化本地文件夹为网站的根目录
  • hexo new [layout] <title>命令用于新建文章，一般可以简写为hexo n
    layout 可选参数，用以指定文章类型，若无指定则默认由配置文件中的 default_layout 选项决定
title 必填参数，用以指定文章标题，如果参数值中含有空格，则需要使用双引号包围
  • hexo s -d 本地启动hexo
    访问http://localhost:4000/admin 进行文章可视化编辑(需安装hexo-admin)
  • hexo clean或hexo c 清理缓存
  • hexo deploy或hexo d 命令用于部署网站
  • hexo clean && hexo deploy清理缓存重新部署

    hexo文章显示部分内容

  • 在要隐藏的内容前添加一行’‘即可，注:不可加在文章最前

Markdown基本语法

"#文字"为标题文字 "#"号数量为对应标题层级
"*文字*"为加粗
三个以上的"-"或"*"为分割线
"![图片alt](图片地址 ''图片title'')"
图片alt就是显示在图片下面的文字，相当于对图片内容的解释
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加
[超链接名](超链接地址 "超链接title"),title可加可不加
"-,+,*"都可以用于无序列表，符号与文字中间需要空格
"`代码`"为单行代码块
多行代码块：
"  ```
  代码
  代码
  ```  "
